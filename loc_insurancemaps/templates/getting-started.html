Project Background →

1. Sign up!

Start by creating an account. Head back here once you are signed in.

If you don't want to sign up, no problem, you can still browse volumes, documents, layers, or web maps that other people have worked on.

2. Find a community you are interested in

Use the list on this page or head to volumes to find a set of maps to work on. Once you select a volume, you will find yourself at the volume summary page.

3. Load the volume

In the volume summary you can load the map sheets into the database. Once loaded, you will see all of the sheets in the "unprepared" list.

If someone else has already loaded the volume, then just pick a sheet to start working with it.

4. Dive in!

In this application, the entire georeferencing process comprises three sequential steps, each with its own tool interface:

  1. Preparation
  2. Georeferencing
  3. Trimming (optional)
These steps need to be completed in order, but they don't have to be completed all in one sitting. In fact, Georeferencing and Trimming can be revisited at any time, so you can improve on your past work, or build from someone else's. Tips for using each tool accompany its interface.
